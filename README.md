# ECG Monitor

ブラウザで動作する心電図モニターのシミュレーター。教育・デモ用途。

タップ操作で以下を切り替えられます:

- **リズム**: 正常洞調律、洞性徐脈/頻脈、心房細動/粗動、PSVT、PAC、WPW、PVC、VT、Torsades、VF、心室固有調律、I〜III度AVブロック、RBBB/LBBB、高/低K血症、STEMI、ペーシング、PEA、心静止
- **心拍数 (HR)**: 0〜260 bpm
- **条件**: ST偏位、QRS幅、PR間隔、T波、基線ノイズ、呼吸性動揺

## 使い方

`index.html` をブラウザで開く、または GitHub Pages 経由でアクセス。

## 免責

教育・デモ目的のシミュレーションです。実際の診療判断には使用しないでください。
